La **matriz de volumen** es una herramienta diseñada para mapear los _canales_ dentro de cada _etapa_ del [_main funnel_](../../fundamentals/vgm-funnel.md).

Para mapear \[_x]_, \[_y]_ y \[z] utilizamos la matriz de volumen.
